# vscode常用快捷键

- F1/Ctrl+Shit+P：万能键，谁用谁知道
- Ctrl + P：文件切换
- Ctrl + 空格：自动提示(输入法下不可行)
- F12/Ctrl + 左键：跳转到定义
- Shift + F12：预览定义
- Ctrl + G：跳转行号
- Ctrl + /：注释切换
- Alt + ↑↓：整行上下移动
- Ctrl + ↑↓：编辑器垂直滚动条上下移动，光标不动
- Ctrl + Backspace/Delete：整词/连续空白删除
- Ctrl + →←：光标整词移动
- Ctrl + F查找/Ctrl+Shift+F在文件中查找
- Shift + Alt + ↑↓：向上下复制当前行
